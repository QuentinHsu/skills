# Skills

Personal AI coding assistant skill collection following the [Agent Skills](https://agentskills.io/) spec.

中文版本：[`README.zh-CN.md`](README.zh-CN.md)

## Install

```bash
pnpx skills add QuentinHsu/skills
```

Or with npm:

```bash
npx skills add QuentinHsu/skills
```

During installation, you can choose the target agent (Cursor, Claude Code, etc.) and scope (project/global). You can also specify an agent explicitly:

```bash
pnpx skills add QuentinHsu/skills -a cursor
```

## Update

If the repository has already been installed, run the same install command again to pull the latest skills and rules:

```bash
pnpx skills add QuentinHsu/skills
```
